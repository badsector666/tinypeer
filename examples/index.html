TinyPeer Examples

Simple, minimal examples demonstrating TinyPeer features.

Basic Connection (Data Channels)

  • Host - Accept connections from other peers
  • Client - Connect to a host

Video Call

  • Caller - Start a video call
  • Callee - Receive a video call

Simple Chat

  • Chat - 1-to-1 peer-to-peer chat

Instructions

  1. Open multiple windows/tabs to test peer connections
  2. Each peer will be assigned a random unique ID when it connects
  3. Copy the peer ID from one window and paste it into the other to establish a connection
  4. For video calls: Open the callee first, copy its ID, then open the caller and paste the callee's ID
